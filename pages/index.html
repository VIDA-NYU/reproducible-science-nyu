ReproZip

ReproZip is a tool that automatically captures provenance of experiments and packs all the necessary ﬁles, library dependencies, and variables to reproduce the results. Reviewers can then unpack and run the experiments without having to install any additional software. Previously, this involved typing 2 commands to pack and then 2 commands to unpack. But now, we've released a graphical interface for unpacking, lowering the barrier EVEN MORE to reproducing another's work.

ReproZip Examples

This website showcases some examples and use cases from different domains that showcase the ReproZip, a tool developed in-house at NYU aimed at simplifying the process of creating reproducible experiments. It automatically captures operating system calls and creates a package that contains all the binaries, files, and dependencies required to run a given command on the original researcher’s computational environment. Find examples from digital humanities, chemical engineering, machine learning, and more in this GitHub repository!
